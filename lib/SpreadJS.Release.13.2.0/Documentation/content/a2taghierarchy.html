    Tag Hierarchy
    In This Topic

    The Spread, Sheet, and Column are the basic elements with tag hierarchy. The three tags are: gc-spread-sheets, gc-worksheet, and gc-column.

    The following example displays the basic tag hierarchy:

    Tag
    Copy Code 
    <gc-spread-sheets>
  <gc-worksheet>
    <gc-column></gc-column>
    ...
  </gc-worksheet>
  ...
</gc-spread-sheets>
