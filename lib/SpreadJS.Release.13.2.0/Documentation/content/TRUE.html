    TRUE
    In This Topic

    This function returns the value for logical TRUE.

    Syntax

    TRUE()

    Arguments

    This function does not accept arguments.

    Data Types

    Does not accept data. Returns numeric (boolean) data.

    Example

    TRUE() gives the result 1 (TRUE)
