    SparklineChanged Event
    In This Topic
    Occurs when the sparkline has changed.
    Syntax
    • Javascript (Usage)
    • Javascript (Specification)
    var instance; // Type: Events
instance.SparklineChanged = function(sheet, sheetName, sparkline) { };
    SparklineChanged = function ( 
   sheet : Worksheet,
   sheetName : string,
   sparkline : Sparkline
) { };

    Parameters

    sheet
    The sheet that triggered the event.
    sheetName
    The sheet's name.
    sparkline
    The sparkline whose property has changed.
    Example
    Removing the sparkline causes a change.
    • Javascript
    var data = new GC.Spread.Sheets.Range(1, 0, 8, 1);
var setting = new GC.Spread.Sheets.Sparklines.SparklineSetting();

setting.options.showHigh = true;
